# assembly-coding
Writing my own compiler and optimising it

This is a "The Un-Serious Durham Programming Language"

It is a turing complete programming language that compiles to x86-64 assembly code.

## Features

- Basic arithmetic operations: addition, subtraction, multiplication, division
- Variable assignments and usage
- Print statements
- Conditional statements (if-else)
- While loops
- For loops
- Vector/Array allocation and access
